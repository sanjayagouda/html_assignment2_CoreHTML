Some HTML Tag names and their definitions

  1. The tag name is "a" & it defines a hyperlink
  2. The tag name is"b" & it Defines bold text
  3. The tag name is"body" & it Defines the document's body
  4. The tag name is"br" & Defines a single line break
  5. The tag name is"footer" & it a footer for a document or section
  6. The tag name is"form" & it Defines an HTML form for user input
  7. The tag name is"input" & it Defines an input control
  8. The tag name is"main" & it Specifies the main content of a document
  9. The tag name is"small" & it Defines smaller text
  10. The tag name is"sup" & it Defines superscripted text
